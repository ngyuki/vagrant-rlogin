# Vagrant TeraTerm Plugin

This is a Vagrant plugin that enables to ssh into vm with TeraTerm.

## Installation

### Vagrant Version Support

```
vagrant plugin install vagrant-teraterm
```

###

## Usage

```
vagrant teraterm
```
